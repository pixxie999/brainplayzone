스도쿠

스도쿠는 9×9 격자에 1~9 숫자를 채우는 논리 퍼즐입니다.

규칙

  • 각 가로 줄(행)에 1~9가 한 번씩만 등장해야 합니다
  • 각 세로 줄(열)에 1~9가 한 번씩만 등장해야 합니다
  • 굵은 선으로 나뉜 3×3 구역에 1~9가 한 번씩만 등장해야 합니다

게임 방법

  1. 빈 칸을 클릭(터치)하여 선택합니다
  2. 하단 숫자 패드 또는 키보드로 숫자를 입력합니다
  3. 잘못된 숫자는 빨간색으로 표시됩니다
  4. 모든 칸을 올바르게 채우면 완료!

2048

2048은 같은 숫자의 타일을 합쳐 2048 타일을 만드는 전략 퍼즐입니다.

규칙

  • 4×4 격자에 숫자 타일이 놓여 있습니다
  • 화살표 키 또는 스와이프로 모든 타일을 한 방향으로 이동합니다
  • 같은 숫자의 타일이 충돌하면 합쳐져 두 배 값이 됩니다
  • 이동할 때마다 빈 칸에 새 타일(2 또는 4)이 생깁니다

게임 방법

  1. PC: 키보드 화살표 키(↑↓←→)로 이동
  2. 모바일: 화면을 스와이프하여 이동
  3. 2048 타일을 만들면 목표 달성!
  4. 더 이상 이동이 불가능하면 게임 오버

점수 계산

타일이 합쳐질 때 합쳐진 값만큼 점수가 올라갑니다. 예: 512+512 합체 → 1024점 추가

플레이 팁

🔢 스도쿠 팁

  • 후보 숫자가 하나뿐인 칸부터 채우세요
  • 같은 행/열/박스에서 이미 있는 숫자를 제외해 나가세요
  • 막힐 때는 힌트 버튼을 활용하세요
  • 키보드 방향키로 셀 간 빠른 이동이 가능합니다

🎯 2048 팁

  • 큰 숫자 타일을 항상 구석(모서리)에 모으세요
  • 이동 방향을 2~3가지로 고정하는 것이 유리합니다
  • 빈 칸을 최대한 확보하면서 진행하세요
  • 타일이 흩어지면 정리하기 어려워집니다
